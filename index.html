Hello! I'm Anisha--a striving web developer and super hero. :) I'm also a graduate of Bard College where I studied Mathematics and completed my senior thesis in combinatorial game theory, a branch of applied mathematics and theoretical computer science that blends combinatorics, game theory and graph theory.

I worked at a project managing consultancy group that was creating the Hackulator: an inexpensive open source graphing calculator for high school students. I worked on this calculator because I strongly believe that every student should have the opportunity to compute and code with a calculator that doesn’t destroy their parents’ budgets.

I soon became fascinated with the programming side of the Hackulator, and was accepted to the Flatiron School's NYC Web Fellowship in Ruby on Rails. Since then I've been working at Flatiron Labs as a Junior Labs Apprentice in Ruby on Rails, helping the team build out their online education presence. It's been a fascinating and whirl-wind experience, and I can't wait to see what happens next!

Petflow: Rails web application for Petflow to display MailChimp and Google Analytics Data.
  • Used MailChimp API to access company's email and Google Analytics data.
  • Applied Cron job to asynchronously load MailChimp data.
  • Created UI to organize data, helping client easily analyze campaigns.
  • Implemented customized tables using HandsOnTable, a JavaScript library.
  • Used D3.js to create bar charts of the data. Deployed app to DigitalOcean using Dokku.
Ruby on Rails, JavaScript, D3.js, AJAX, MailChimp API, HTML, CSS

Twidder: Rails web application for kids to correct celebrity’s tweets. Used Nokogiri to identify top celebrity handles.
  • Implemented Twitter gem to authenticate Twitter API and load top celebrities’ tweets.
  • Used Nokogiri to identify top celebrity handles.
  • Worked with Gingerice, SwearJar, and FFI-Aspell gems to identify misspelled tweets and generate suggested corrections.
  • Applied custom CSS to mimic the Twitter design
Ruby on Rails, JavaScript, D3.js, AJAX, MailChimp API, HTML, CSS

Game of Life: Sinatra web application for the Game of Life using AngularJS and Sinatra.
  • Applied algorithms based on Conway’s Game of Life to represent cells as two dimensional arrays.
  • Mitigated performance problems by chunking requests to the Sinatra API.
  • Note: The Game of Life takes a bit of time to load because it's a lot of cells #codetolearn)
Sinatra, Sessions, AngularJS
